# Demo Script

1. Load the unpacked extension from `apps/extension/dist`.
2. Open ChatGPT.
3. Type a broad prompt, for example:

   `Tolong bantu saya untuk membuatkan sebuah penjelasan yang lengkap dan detail tentang API Gateway, Swagger, dan cara mengkomunikasikan endpoint ke kelompok lain.`

4. Show the Gnomon widget.
5. Expand the widget and show score, input tokens, output range, total range, issues, and suggestions.
6. Click Optimize Prompt.
7. Show before/after comparison and token savings.
8. Click Apply Optimized Prompt.
9. Open the popup and show token budget, language, local-only mode, and optimization mode.
