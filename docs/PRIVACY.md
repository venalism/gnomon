# Privacy

Gnomon MVP is local-only.

- Prompt text is analyzed in the browser.
- Prompt text is not sent to external APIs.
- Prompt text is not stored by default.
- Settings are stored through Chrome storage APIs.
- Host permissions are limited to ChatGPT domains.
- The MVP does not include AI rewrite, analytics, cloud sync, auth, or prompt history.

If AI rewrite is added later, it should be opt-in and clearly disclose what text leaves the browser.
