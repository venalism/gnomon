# Gnomon PRD Summary

Gnomon is a Chrome/Chromium browser extension that predicts AI prompt token usage and helps users write clearer, cheaper, and more controlled prompts before sending them to an AI chatbot.

## MVP Definition

- Support ChatGPT web only.
- Analyze prompt text locally.
- Show estimated input tokens, predicted output token range, total token range, confidence, and prompt score.
- Detect common prompt issues.
- Suggest prompt improvements in English and Indonesian.
- Generate a rule-based optimized prompt.
- Let users compare, copy, and apply the optimized prompt.
- Store settings locally.

## Non-Goals

- No backend.
- No login.
- No cloud sync.
- No prompt history by default.
- No AI-powered rewrite in MVP.
- No claim of exact token billing accuracy.
